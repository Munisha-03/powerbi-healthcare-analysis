# Healthcare Center Performance Dashboard (Power BI)

## 📊 Project Overview
This project analyzes the performance of a healthcare center using an interactive Power BI dashboard. It focuses on financial performance, patient insights, provider analysis, and overall trends.

The dashboard is built using multiple interconnected datasets (2440 records) to provide meaningful insights for decision-making.

---

## 📁 Dashboard Pages

### 1. Main Page
- Overview of key metrics
- Total Revenue
- Total Patients
- Average Satisfaction Score
- Quick summary visuals

---

### 2. Financial Overview
- Revenue analysis across time
- Financial performance breakdown
- Service-wise revenue distribution

---

### 3. Provider Insights
- Performance of individual providers
- Revenue per provider
- Patient count by provider
- Comparative analysis

---

### 4. Trend Analysis
- Monthly trends of patients and revenue
- Pattern identification over time
- Growth and decline insights

---

### 5. Provider Tooltip (Drill-through Feature)
- Detailed view of selected provider
- Shows:
  - Total revenue
  - Total patients
  - Individual performance metrics
- Improves interactivity and user experience

---
